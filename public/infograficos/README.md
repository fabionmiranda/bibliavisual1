# Infográficos Bíblicos

Pasta destinada ao armazenamento de infográficos estáticos (JPG ou PNG).
O sistema busca automaticamente arquivos com o nome do ID do livro (ex: genesis.jpg, levitico.png).

Como carregar novos infográficos:
1. Salve sua imagem no formato JPG ou PNG.
2. Nomeie o arquivo com o ID do livro em letras minúsculas (ex: mateus.jpg).
3. Cole o arquivo nesta pasta.
